## 🇺🇿 O'zbek

### Issue qoldirish

- Xato topilsa → **Bug report** templatedan foydalaning
- Yangi komponent taklif qilsangiz → **Feature request** templatedan foydalaning
- Savol bo'lsa → **Discussions** bo'limidan foydalaning

### Commit convention

```
feat: yangi funksiya
fix: xato tuzatish
docs: dokumentatsiya
style: formatlash
refactor: refaktoring
test: test qo'shish
```

## 🇷🇺 Русский

### Создание Issue

- Нашли баг → используйте шаблон **Bug report**
- Хотите новый компонент → используйте шаблон **Feature request**
- Есть вопрос → используйте раздел **Discussions**

### Соглашение по коммитам

```
feat: новая функция
fix: исправление ошибки
docs: документация
style: форматирование
refactor: рефакторинг
test: добавление тестов
```

## 🇬🇧 English

### Creating an Issue

- Found a bug → use the **Bug report** template
- Want a new component → use the **Feature request** template
- Have a question → use the **Discussions** section

### Commit convention

```
feat: new feature
fix: bug fix
docs: documentation
style: formatting
refactor: refactoring
test: adding tests
```
